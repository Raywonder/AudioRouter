ASIOA Audio Router Release Agreement

This installer contains the ASIOA Windows control app, native ASIOA driver
package, route model, feedback-guard verification tool, accessibility support
files, release notes, and documentation.

Driver installation:
The setup wizard can register the packaged ASIOA virtual ASIO driver for local
testing when ASIOA.Driver.dll and the driver install script are included in the
installer. You may install the driver now, be reminded later, or install only
the control panel. Driver registration requires administrator permission.

Important current limitations:
This package is intended for release evaluation, development, and hands-on local
driver testing. The packaged native driver is not yet a publicly signed
production driver. Kernel-level audio capture, per-application system audio
capture, and VST3 or CLAP bridge plug-ins are not complete production features
in this build. Those components require additional implementation, licensing
review where applicable, compatibility testing, and driver or code signing
before broad public distribution.

By installing, you agree that:

1. The software is provided as-is for release evaluation and development.
2. The installer may register the packaged ASIOA virtual ASIO driver when you
   choose the driver installation option and the driver files are present.
3. The current app can test and display the route model, feedback guard, channel
   layout, buffering settings, startup settings, driver health, and accessibility
   surface.
4. The current package is not yet a finished, publicly signed virtual ASIO audio
   device release for production use.
5. Any public ASIO driver distribution must follow either Steinberg's GPLv3 ASIO
   SDK path or a reviewed proprietary ASIO license path, plus the applicable
   Windows driver and code-signing requirements.
6. Accessibility features are under active development and should be tested with
   NVDA, Narrator, and keyboard-only workflows before production use.
