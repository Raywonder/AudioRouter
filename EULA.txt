ASIOA Audio Router License Agreement

This installer contains the ASIOA Audio Router Windows control application,
ASIO host driver package, audio-routing configuration tools, feedback guard,
accessibility support files, release notes, and documentation.

ASIOA is designed to help route audio between Windows audio devices,
ASIO-capable hosts, DAWs, virtual buses, accessibility audio paths, and
supported audio applications. Some features require administrator permission
because Windows audio driver registration and driver updates are system-level
operations.

Driver and endpoint installation:

The installer may register or update the packaged ASIOA ASIO driver and, when
included in a release package, the ASIOA Windows audio endpoint driver package.
The ASIO driver is used by ASIO-capable hosts. The Windows audio endpoint
driver package is used by applications that select normal Windows speaker and
microphone devices through WDM, WASAPI, DirectSound, or related Windows audio
APIs.

When an existing ASIOA version is running, the installer may ask it to stop,
disconnect active driver sessions, update files, register updated driver
components, and then allow the user to launch ASIOA again. Active audio may
pause or restart during installation or driver updates.
